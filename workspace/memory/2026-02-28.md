### Lesson: Docker Restart Commands
The gateway runs in Docker on DigitalOcean. When telling Sam to restart:
- **Quick restart:** `docker restart openclaw-openclaw-gateway-1`
- **Full restart:** `cd /root/openclaw && docker compose down && docker compose up -d`
- **Never** just say `docker compose restart` without specifying the directory — the compose file lives at `/root/openclaw/docker-compose.yml`
